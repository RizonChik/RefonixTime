Плагин для Minecraft 1.16.5 - показывает время в городах России с красивыми RGB градиентами.

## Команды

- `/time <город>` - время в городе
- `/time <город> <+/-часы>` - добавить город со смещением от МСК

**Примеры:**
```
/time москва
/time самара +1
/time калининград -1
```

## Встроенные города

Москва, Екатеринбург, Новосибирск, Владивосток, СПб

## Установка

1. Скачай jar файл
2. В папку `plugins/`
3. Перезапуск сервера

## Требования

Minecraft 1.16.5 + Bukkit/Spigot/Paper
